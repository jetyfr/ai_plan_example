## Solicitud de aplicación

Quiero hacer una aplicación web 100% en cliente para gestionar tareas:

- Las tareas tienen un título, una descripción y el instante de creación.
- Cada tarea va en una tarjeta y se puede editar, borrar o marcarla como realizada. Marcarla como realizada debe apreciarse bien a nivel visual.
- Las tarjetas que se crean van a un canvas donde se pueden colocar a gusto del usuario y para cada tarjeta se puede configurar el color de entre 8 colores alternativos.

## Requisitos técnicos y de diseño

- Quiero una aplicación web con aspecto moderno y un diseño elegante implementada usando HTML, CSS y vanilla JavaScript, con las librerías que consideres necesarias, pero sin usar frameworks como React, Vue o Angular.
- La aplicación debe estar adaptada para su correcto funcionamiento en PC o móvil.
- El estado de la aplicación se guardará en localStorage.
